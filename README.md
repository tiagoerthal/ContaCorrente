# Conta Corrente

![](https://imgur.com/dXE24Wj.gif)

## Introdução

Um código que se assemelha com uma conta bancária feito para testar a orientação a objetos.

## funcionalidades

- Capaz de sacar dinheiro, depositar o valor selecionado, mostrar o saldo atual da conta, com a capacidade de mostrar o extrato completo da conta e transferir para outra conta.
- Proíbe o usuario de sacar, depositar valores negativos ou valores que não coincidem no saldo.
